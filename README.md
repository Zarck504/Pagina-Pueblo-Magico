# Página Turística de Hidalgo

¡Bienvenido al repositorio de la página web turística de Hidalgo! Este proyecto tiene como objetivo mostrar la belleza, cultura y tradiciones del estado de Hidalgo, México.

## Características

- Diseño moderno y responsivo
- Secciones destacadas de lugares turísticos
- Información sobre gastronomía local
- Galería de imágenes
- Formulario de contacto

## Estructura del Proyecto

```
Pueblo Magico Hidalgo/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos principales
└── img/               # Directorio para imágenes
    ├── hero-bg.jpg    # Imagen de fondo del encabezado
    ├── huasca.jpg     # Prismas Basálticos
    ├── real.jpg       # Real del Monte
    └── tula.jpg       # Zona Arqueológica de Tula
```

## Cómo usar

1. Clona este repositorio
2. Abre el archivo `index.html` en tu navegador web
3. Explora la página para descubrir los atractivos de Hidalgo

## Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a Internet (solo para fuentes de Google)

## Personalización

Para personalizar la página:

1. Reemplaza las imágenes en la carpeta `img/` con tus propias fotografías
2. Modifica los colores en el archivo `css/styles.css` (variables en la parte superior)
3. Actualiza el contenido en `index.html` según sea necesario

## Créditos

- Fuentes: Google Fonts (Poppins)
- Iconos: Font Awesome (incluido en el HTML)
